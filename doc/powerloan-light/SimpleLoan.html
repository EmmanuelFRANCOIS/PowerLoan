Properties:
Name Type Description
id integer

loan id. Auto increment (from db)
uuid string

unique identifier of user/device. Generated with uuid JS library. Default : ""
title string

title of the loan. Default : "My Simple Loan"
description string

description of the loan. Default : ""
destination string

loan's destination : "realEstate", "car", "consumer", "student", "business" or "other". Default : "realEstate"
destinationOther string

custom loan's destination when destination is "other". Default : ""
active boolean

Real contracted loan or simulation. True if the loan is a real contracted loan. False if the loan is only a simulation. Default : false
calculationMode string

Define the result we want to get : "capital", "duration", "interest", "repayment". Default : "repayment"
capital integer

borrowed capital ($). Default : 100000
releaseDate date

date of capital release. Default : 1rst of next month from today at 08:00:00
beginDate date

date of loan beginning (first repayment). Default : releaseDate + one month
endDate date

date of loan ending (last repayment). Default : beginDate + (durationTime - 1) (in months)
durationPeriods integer

number of period of repayment. Default : 120
durationTime integer

duration of loan in months. Default : 120
interestRate decimal

raw annual interest rate. Default: 0.0350
repayment decimal

periodic repayment ($) (only if constant repayment. If variable repayment, it must be calculated for each period.). Default : 1000
lender string

lender name. Default : ""
lenderContact string

lender's contact name. Default : ""
lenderAddress string

lender's address. Default : ""
lenderContactPhone string

lender's contact phone number. Default : ""
lenderContactEmail string

lender's contact email. Default : ""
lenderContractRef string

reference of the loan contract. Default : ""
lenderContractDate date

date of loan contract. Default : null
createdAt date

date of loan creation. Default : Now
modifiedAt date

date of last loan modification. Default : Now
